# 固定条组件

> 固定条组件 `fixbar` 是指当滚动条滑动时，始终固定在页面一侧的工具条元素，除了内置的 `TOP` 条功能外，还可以灵活地扩展任意自定义的工具条。在 `2.x` 版本中，`fixbar` 属于 `util` 模块的子集。
